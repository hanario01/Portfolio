# Portfolio Site

## 概要
本サイトは、私自身の制作物をまとめた **ポートフォリオサイト** です。  
Web制作における **構造設計・UI表現・JavaScriptによる挙動実装** を中心に、  
「実務を想定した構成」を意識して制作しています。

---

## コンセプト
**「落ち着き・余白・没入感」**

視覚的な派手さよりも、
- 情報の伝わりやすさ
- スクロール体験
- セクションごとの世界観の切り替え  

を重視し、  
閲覧者が自然に作品へ集中できる構成を目指しました。

---

## 使用技術
- PHP  
  - 共通パーツ（header / footer）の分割管理  
  - BASE_URL を用いたパス管理
- HTML5  
  - セマンティック構造を意識したマークアップ
- CSS3  
  - CSS変数によるテーマ管理  
  - セクション単位のスタイル設計  
  - 擬似要素・アニメーションを用いた装飾
- JavaScript（Vanilla JS）  
  - ハンバーガーメニュー制御  
  - スクロールイベントによるUI変化  
  - Works セクションの画像切り替え  
  - アニメーション制御

---

## 実装している主な機能
- スクロールに応じたヘッダー表示切り替え
- ハンバーガーメニュー（SP対応）
- WORKS セクションのメイン画像切り替え
- 画像リビールアニメーション
- セクションごとの背景色トランジション
- ABOUT / CONTACT セクションの演出表現

---

## 設計で意識した点
- 実務を想定し、**共通要素はPHPで分割**
- クラス名は「見た目」ではなく「役割」ベース
- JavaScript用クラスとスタイル用クラスを分離
- デザイン要素が強い箇所でも、拡張しやすい構造を意識
